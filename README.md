## Projeto

ecommerce_app é uma aplicação de comércio eletrônico em desenvolvimento usando Flutter ,Firebase e o Padrão BloC. 

Consiste no desenvolvimento de um ecommerce, desde o planejamento até o projeto, desenvolvimento e testes. 

## Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:

- Dart
- Flutter
- Firebase
- BLoC
